前言

山姆鍋在使用 Surface Pro 的 Windows 系統時幾乎確定一定會開啟 Ubuntu 環境的 Shell。為了方便操作，自然希望 Windows Terminal 一開始就是進入 Ubuntu 環境。由於也會使用 Zsh + Oh My Zsh 作為操作介面，也需要安裝設定 Nerd fonts for power line。
